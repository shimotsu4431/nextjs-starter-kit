## 環境構築
```bash
npm install
# or
yarn
```

node および yarn のバージョンはバージョン管理ツール [Volta](https://volta.sh/) を利用し、以下で固定しています。

```
"node": "16.13.1"
"yarn": "1.22.17"
```

## 開発サーバー立ち上げ
```bash
npm run dev
# or
yarn dev
```
